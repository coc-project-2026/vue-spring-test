## 1. サービスの概要 (Overview)
- **目的**: ログイン後、単語帳を通じて言語学習を支援するために
- **対象範囲**: 単語の登録、出力、修正、削除

## 2. 機能要件 (Functional Requirements)
※ 実装予定の最小限の機能を記載してください。

- **[機能1]**: 単語の登録・削除機能
  : 新しい単語を登録すると削除する機能
- **[機能2]**: 単語修正機能
  : 登録された単語を修正する機能
- **[機能3]**: 登録した単語一覧の表示機能
  ：登録された単語リストをページごとに10個ずつ表示する機能
- **[機能4]**: ページネーション機能

## 3. データ定義 (Data Definition)

- **ログイン情報**: メールアドレス, パスワード, アクセストークン
- **単語情報**: ID, 単語, 意味

## 4. 画面遷移 (UI Flow)
- **ログイン画面**: メールアドレスとパスワードを入力してログインする画面
- **メイン画面**: 単語帳ページへ遷移するボタンを表示する画面
- **単語帳画面**: 単語の登録、修正、削除、表示機能
- **エラー画面**:   404 Not Found エラー、403 Forbidden エラー、500 Internal Server Error、サーバー通信エラー
