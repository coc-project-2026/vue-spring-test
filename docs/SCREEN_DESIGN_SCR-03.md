# 🎨 画面詳細設計 (SCR-03: 単語帳画面)

## 1. 画面概要
- 単語帳の表示、追加、修正、削除できる画面

## 2. UI要素定義 (UI Elements)
| ID | 要素名 | 種類 | 動作・備考 |
|:---|:---|:---|:---|
| BTN-01 | ログアウトボタン | Button |  クリック時、ログアウト処理を実行しログイン画面(SCR-01)へ遷移 |
| BTN-02 | 追加ボタン | Button | 新規単語を追加 |
| BTN-03 | 編集ボタン | Button | 選択した単語の内容を編集 |
| BTN-04 | 削除ボタン | Button | 選択した単語を削除 |

## 3. 主要コンポーネント (Components)
- **`MenuView`**: ログイン後に表示されるメニュー画面全体を管理するコンポーネント。

## 4. 画面ロジック (UI Logic)
- **単語一覧表示**: APIから取得したデータをリスト形式で表示する
- **単語追加**: 登録ボタン押下時にAPIへ送信する
- **単語編集**: 編集対象の単語を選択する、更新ボタンでAPI送信する
- **単語削除**
