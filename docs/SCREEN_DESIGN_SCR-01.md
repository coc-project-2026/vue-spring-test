# 🎨 画面詳細設計 (SCR-01: ログイン画面)

## 1. 画面概要
- ユーザーがシステムにログインするための画面です。

## 2. UI要素定義 (UI Elements)
| ID | 要素名 | 種類 | 動作・備考 |
|:---|:---|:---|:---|
| INP-01 | メールアドレス入力欄 | Input | ログインメールアドレス入力 |
| INP-02 | パスワード入力欄  | Input | ログインパスワード入力 |
| BTN-01 | ログインボタン | Button | クリック時のログインAPI呼び出し  |

## 3. 主要コンポーネント (Components)
- **'View'**: 画面


## 4. 画面ロジック (UI Logic)
- **初期表示**: ログイン画面を表示する（特別なAPI呼び出しなし）
- **ログイン処理**: ログインボタン押下時、`POST /api/auth/login` を呼び出し、認証を実行する
- **認証成功時**: 取得した accessToken を保存する
- **ユーザー情報取得**: `GET /api/me` を呼び出し、ログインユーザー情報を取得する
- **認証失敗時**: 401/403 エラー時にログアウト処理を実行し、ログイン画面へ遷移する
